# MERN Task Manager App

## Setup (local MongoDB)
1. Ensure MongoDB is installed and running locally (`mongod`).
2. Backend:
   ```bash
   cd server
   npm install
   npm run dev
   ```
3. Frontend:
   ```bash
   cd client
   npm install
   npm start
   ```
Frontend runs on http://localhost:3000 and backend on http://localhost:5000.
